# Image Background Remover - MVP 需求文档

## 项目概述

**项目名称：** Image Background Remover
**项目版本：** v0.1.0 (MVP)
**项目类型：** Web 应用
**开发框架：** Next.js 14 + TypeScript + Tailwind CSS
**部署平台：** Cloudflare Pages / Vercel
**目标用户：** 需要快速去除图片背景的用户

---

## 1. 产品愿景

### 1.1 核心价值
为用户提供一个简单、快速、免费的图片背景去除工具，无需专业技能，上传图片即可获得透明背景的 PNG 图片。

### 1.2 目标用户
- **电商卖家：** 快速制作产品白底图
- **设计师：** 快速去除素材背景
- **内容创作者：** 优化图片素材
- **普通用户：** 制作证件照、头像等

### 1.3 核心优势
- ✅ 零学习成本，上传即用
- ✅ 无需下载安装软件
- ✅ 处理速度快（2-5秒）
- ✅ 完全免费（基于 Remove.bg 免费额度）
- ✅ 隐私安全（图片不存储）

## 2. 功能需求

### 2.1 核心功能（P0 - 必须实现）

**优先级：** P0
**描述：** 支持用户上传需要处理的图片

**详细需求：**

### 4.3 核心 API

#### POST /api/remove-bg
**功能：** 调用 Remove.bg API 去除背景

**请求：**
- Method: POST
- Content-Type: multipart/form-data
- Body: image 文件

**响应：**
- 成功：PNG 图片（透明背景）
- 失败：JSON 错误信息

**示例：**
```bash
curl -X POST \
  -F "image=@photo.jpg" \
  http://localhost:3000/api/remove-bg \
  --output result.png
```

## 5. 数据流

### 5.1 用户操作流程
```
1. 用户访问页面
   ↓
2. 拖拽或点击上传图片
   ↓
3. 前端验证文件（类型、大小）
   ↓
4. 显示预览图
   ↓
5. 用户点击"Remove Background"
   ↓
6. 前端调用 /api/remove-bg
   ↓
7. API 转发到 Remove.bg
   ↓
8. 返回处理后的 PNG
   ↓
9. 显示结果 + 对比滑块
   ↓
10. 用户下载或重新处理
```

### 5.2 API 调用流程
```
Client → Next.js Edge Function → Remove.bg API
        ↓
        Forward request with API Key
        ↓
        Return processed image
        ↓
Client ← Processed PNG ← Remove.bg API
```
